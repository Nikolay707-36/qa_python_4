В проекте реализовано 10 тестов для класса `BooksCollector`.

1. `test_add_new_book_success` — проверка успешного добавления книги.
2. `test_add_new_book_validation` — параметризованный тест валидации имени книги (пустая строка, превышение 40 символов, корректное имя).
3. `test_add_duplicate_book` — проверка защиты от добавления дубликатов.
4. `test_set_book_genre_success` — установка допустимого жанра.
5. `test_set_book_genre_invalid_genre` — попытка установить недопустимый жанр (отсутствующий в `self.genre`).
6. `test_get_books_with_specific_genre` — получение списка книг по заданному жанру.
7. `test_get_books_for_children` — фильтрация книг для детей (исключение жанров с возрастным рейтингом).
8. `test_add_book_in_favorites` — добавление в избранное и защита от дублей в списке избранного.
9. `test_delete_book_from_favorites` — удаление книги из избранного.
10. `test_get_list_of_favorites_books` — получение списка избранных книг.

- Используется `setup_method` для изоляции тестов (чистый экземпляр класса перед каждым тестом).
- Применена параметризация (`@pytest.mark.parametrize`) для сокращения дублирования кода при валидации.
- Все тесты запускаются без ошибок (`pytest -v tests.py`).
